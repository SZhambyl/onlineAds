Функциональный сайт онлайн объявлений с недоработанным дизайном.
В нем есть:
- Регистрационная форма, которая проверяет на правильность через Java Script. 
- Можно подать объявление даже без регистраций. После того как объявление подано, админ решает выставлять товар или нет.
- Есть возможность отправлять сообщение между прадавцом и покупателем, если они оба зарегистрированы на сайт. 
- Работает пойсковик через Ajax. 
- А также можно редактировать свой товар, если пользователь зарегистрирован.
